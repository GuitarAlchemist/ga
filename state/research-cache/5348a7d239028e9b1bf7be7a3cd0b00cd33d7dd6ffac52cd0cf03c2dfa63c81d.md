# Anthropic Claude Model Pricing & Release Dates (July 2, 2026)

## Claude Opus 4.8

- **Pricing:** $5 per million input tokens, $25 per million output tokens
- **Source:** "System Card: Claude Opus 4.8 May 28, 2026" (anthropic.com) and official announcement "Introducing Claude Opus 4.8"

- **Source:** System Card metadata "Claude Sonnet 5 June 30, 2026" and official announcement "Introducing Claude Sonnet 5"
