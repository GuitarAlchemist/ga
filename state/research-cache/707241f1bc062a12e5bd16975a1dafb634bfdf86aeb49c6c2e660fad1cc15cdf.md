## Claude Opus 4.8
- Release date: May 28, 2026
- Pricing: $5 per million input tokens, $25 per million output tokens
- No introductory pricing mentioned

## Claude Sonnet 5
- Introductory pricing: $2 per million input tokens, $10 per million output tokens through August 31, 2026
- Standard pricing (September 1, 2026 onwards): $3 per million input tokens, $15 per million output tokens
- Introductory pricing expiration: August 31, 2026
- Batch API pricing (introductory): $1 per million input tokens, $5 per million output tokens

## Claude Haiku 4.5
- Pricing: $1 per million input tokens, $5 per million output tokens
- No introductory pricing mentioned
- Available to all users

Note: Claude Opus 4.8 maintains the same pricing as Opus 4.7 and earlier Opus 4.x models ($5/$25). Sonnet 5 has introductory pricing lower than Sonnet 4.6 ($3/$15). Haiku 4.5 maintains the same pricing as Haiku 3.5.
