The scale pattern panel

 

In the bottom part of the scale pattern panel, you see on a staff the notes of the selected scale for the current pattern. You can move your mouse over the notes, and you'll see that the cursor will move accordingly on the fingerboard to show you the position of the note.

The number below each notes represent the interval quality of the note. Underlined degrees indicate a primary character tone while italic ones indicate a secondary character tone (see character tones).

The scale can be displayed using both a sharp key and a flat key in certain cases, and sometimes in one of theses only. You can click on the tabs to switch between the sharp key and flat key staves. When a key is not available, the tab is marked with a red cross, and a green check mark otherwise.

 

The staves group box

 

The stave group box is used to change the way you see the scale pattern on the staves.

The minor key check box can be used to change the key into a minor key.
The key alterations is useful to show the alterations due to key directly on the staff.

When the derivative approach check box is on (by default), Guitar Alchemist computes the key that suits the best for writing the mode, computing the parent scale from which the mode is derived. You can turn it off if you want to use the same key as the root of the mode. Note that when this option is on, the third status panel will indicate the parent scale for this mode this is also the name of the key which is used on the staff.

 

The fingering pattern group box

 

You can change the fingering pattern number to climb up and down the fingerboard. Note that you can also change it directly from the fingerboard by just clicking on one of the scale positions outside the pattern (Visible only when the Display non pattern option is turned on)

Checking the extended check box, you can switch between the condensed and expanded hand shape. Neither approach is better than the other, they are simply different. It's up to you to experiment to see which approach you prefer.

The octave check box is used to display the pattern after the 12th fret. In some situations, clicking on it won't have any effect because there's no octave position for the position you are.
