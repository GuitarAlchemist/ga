The chord dictionary panel

 

 

The chord dictionary panel contains :

A tool bar to control which chords are listed in the dictionary.

An information panel which indicates the current chords (and optionally the name of the first note if the selected voicing is an inversion). The number of chords in the list is also indicated.

The actual list of chords which can be presented in a simple or detailed list (see detailed view)

 

The chord dictionary tool bar:

 

These buttons allows you to switch between the simple and detailed view (The detailed view is detailed further).

 

These buttons allow you to set the maximum difficulty for the voicings listed in the chord dictionary (Easy, medium, hard, very hard, mortal).

 

These buttons controls which kind of inversions you want to be included in the list of chords :

* means any inversion

R means the root voicing only

1 means the first inversion only

2 means the second inversion only

3 means the third inversion only (This one is useful for 7th chords and their extensions only)

 

The first button allows you to exclude chords containing muted strings from the list.

The second button is used to exclude chords which contain open strings positions from the list.

The third button excludes barré chords from the list.

 

The detailed view:

 

 

The detailed view is used when you want to find a specific chord voicing. The main advantage of using this detailed view resides in the fact that you can sort the chords by their characteristics. You can change the sorting order just by moving the columns to the left. For example, if you want your list to be sorted by contrast, then by difficulty, move the contrast column on the leftmost, then move the difficulty column just after the contrast column. You can also reverse the order just by clicking on the column header. Note that the selected voicing in simple and detailed view always remain synchronized.

This feature is not available in the free version of Guitar Alchemist.
