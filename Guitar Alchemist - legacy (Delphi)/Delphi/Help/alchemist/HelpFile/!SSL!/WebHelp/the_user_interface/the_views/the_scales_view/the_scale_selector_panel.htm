The scale selector panel

 

The scale root group box is used to select the root of the scale you want to display. You can double click on an item to play the note.

 

The scale name group box features a scale tree selector with contains a parent scale node that contains all the modes as children nodes. The number indicates the rank of the mode toward the scale (For example, the Dorian mode is the second mode of the Major scale). The numbers beside the mode name represent each degrees of the scale.

 

The lock key check box is used to help guitarists finding chord vamps to play over a given mode (This feature is not included in the free version of Guitar Alchemist).

 

Example:

Imagine you want want to find a small series of chords that you can play over B Dorian mode.

- Select B as scale root and Dorian as scale name, then click on the Lock key check box.

- Select a chord template that sounds good to you, Bm7 for example and choose a fingering in the chord dictionary.

- Now pick another scale (usually, a second, fourth or fifth further sounds good, respectively phrygian, mixolydian and aeolian if you start from the Dorian mode), mixolydian for example.

- Select a template that sounds good to you for this mode, Edom9 for example and choose a fingering in the chord dictionary.

 

It's that simple. You've found a good vamp (or simple series of chords) to play with the B Dorian mode !
