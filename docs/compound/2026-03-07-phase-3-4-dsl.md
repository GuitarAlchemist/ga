## Proposed Promotions (fsharp-architect)

### P1 — `Map.ofList` schemas → `schemaOf` + `invokeWith` (Layer 2, Domain)

**File**: `Common/GA.Business.DSL/Closures/GaClosureRegistry.fs`

```fsharp
module SchemaBuilder =
    type SchemaBuilder = { items: (string * string) list }
    let schemaOf = { items = [] }
    let param name desc schema = { schema with items = (name, desc) :: schema.items }
    let build schema = schema.items |> List.rev |> Map.ofList

let invokeWith (registry: GaClosureRegistry) (name: string) (inputs: (string * obj) list) =
    registry.Invoke(name, Map.ofList inputs)
```

**Before**: `InputSchema = Map.ofList [ "symbol", "string"; "semitones", "int" ]`
**After**: `InputSchema = schemaOf |> param "symbol" "string" |> param "semitones" "int" |> build`

Risk: **low** — purely additive, old `Map.ofList` still works.

### P2 — F# Result unwrapping → `FSharpResultExtensions.Match` (Layer 1, Core)

**File**: `Common/GA.Core/Functional/FSharpInterop.cs` (new)

```csharp
public static TOut Match<T, TError, TOut>(
    this FSharpResult<T, TError> result,
    Func<T, TOut> onSuccess,
    Func<TError, TOut> onFailure)
    => result.IsOk ? onSuccess(result.ResultValue) : onFailure(result.ErrorValue);
```

**Before**: 4-line `if (result.IsOk) { ... } else { ... }` block repeated 12+ times
**After**: `result.Match(ok => ..., err => ...)`

Risk: **low** — pure extension method, zero runtime cost.

### P4 — Completion item literals → `mkKeywordCompletion` / `mkFunctionCompletion` (Layer 3, DSL)

**File**: `Common/GA.Business.DSL/LSP/CompletionProvider.fs`

```fsharp
let mkCompletion label kind detail doc insertText =
    { Label = label; Kind = kind; Detail = detail; Documentation = doc; InsertText = insertText }

let mkKeywordCompletion label detail doc =
    mkCompletion label CompletionItemKind.Keyword (Some detail) (Some doc) (Some label)

let mkFunctionCompletion label detail =
    mkCompletion label CompletionItemKind.Function (Some detail) None (Some label)
```

**Before**: 5-field record literal × 110
**After**: `mkKeywordCompletion "maj7" "Major 7th chord" "Major triad + M7. Intervals: R-M3-P5-M7."`

Risk: **low** — migrate incrementally per completion group.
