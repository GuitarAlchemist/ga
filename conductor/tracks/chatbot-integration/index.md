# Chatbot Intelligence Integration

This track focuses on integrating the newly created "Harmonic Intelligence" services from `GA.Business.ML` into the `GaChatbot` application.

## Goals
- Enable **Spectral RAG** in the chatbot (weighted retrieval).
- Enable **Modal Flavor** awareness (Lydian/Phrygian explanations).
- Enable **Wavelet** analysis for progression queries (future).
