# NOAA Science On a Sphere Milky Way Attribution

Prime Radiant keeps `public/textures/milky-way-nasa-sos-2048.jpg` as the fallback Milky Way sky panorama.

- Catalog: https://sos.noaa.gov/catalog/datasets/milky-way-panorama/
- Runtime source file: https://sos.noaa.gov/ftp_mirror/astronomy/milky_way/all_sky/2048.jpg
- Dataset contact/developer: NASA / Goddard Space Flight Center (GSFC)
- Data source: Axel Mellinger
- Retrieved: 2026-05-24

This replaced the previous Solar System Scope Milky Way texture because it is a real all-sky Milky Way panorama intended for spherical projection. The preferred high-resolution tier now uses the ESO GigaGalaxy panorama.
